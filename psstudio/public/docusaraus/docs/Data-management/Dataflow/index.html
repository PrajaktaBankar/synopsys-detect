Data flow

What is data flow?â€‹

Flow based tool for programming, used for fetching, transforming, and storing data in PredictSense. It can be used for creating the logical flow for any type of data. Data flow helps to process the unstructured file in to structured file.

Types of data flowâ€‹

  1. Interceptor data flow
  2. Direct data flow

Data flow nodesâ€‹

  1. ps_inject_node - It is mendatory to add to create any flow.
  2. ps_savedata - Savedata node will create a new file whenever the flow get executed.
  3. ps_appenddata - Appenddata node will create one processed file in dataset page and one downloadable file in data connection module. It will append the data in same file whenever the flow get executed.
  4. python3-function - It is used to write the code as per the use case. below is sample code.

Interceptor data flowâ€‹

You can attach a data flow to a data connection, so whenever the data connection is executed it will be passed through the data will generate the output. This type of flow is called as interceptor flow. After creating a connection, you can click on the data flow option to create a interceptor data flow.

Steps to create and execute a interceptor data flowâ€‹

  1. Goto data->Data connection.
  2. Create connection for any data source like URL, MSSQL, MySQL, PostgreSQL, Snowflake, Google Bigquery, SFTP, Amazon S3.
  3. Click no button to create the interceptor flow.
  4. Drag and drop the necessary node to create the flow like ps_inject_node, ps_savedata, ps_appenddata and use python3-functon to write the code.
  5. Connect the node like ps_inject_node->python3-function->ps_savedata (for any flow the inject node is mandatory to add).
  6. Click on create button.
  7. Execute the flow from data connection page. Click on source and it will open the popup window were you can start the flow execution by clicking on execute data flow button and you can see the saved data in dataset module.

Python3-function code for interceptor flowâ€‹

Refer the below added sample code to create the flow as per the use case. Below code is used for removing temperatureMax column from data and saving new file.

  import pandas as pd
  import json
  df = pd.read_csv(msg['filePath'])
  df.drop('temperatureMax',inplace = True, axis =1)
  df = df.head(10)
  json_data = df.head().to_dict()
  msg['filePath']=("savedata.csv")
  msg['payload'] = json.dumps(df.to_dict(orient='records'))
  print(msg)
  return msg

Direct data flowâ€‹

Whenever we need to transform some data which is not coming from data connections we can use direct data flow.

Steps to create and execute a direct data flowâ€‹

  1. Goto Data->Data Flow
  2. click on Create now or create button
  3. Enter the flow name, select folder and select scheduler
  4. click on Next button
  5. Drag and drop the necessary node to create the flow like ps_inject_node, ps_savedata, ps_appenddata and use python3-functon to write the code
  6. Connect the node like ps_inject_node->python3-function->ps_savedata (for any flow the inject node is mandatory to add)
  7. Click on create button
  8. click on start data flow button. It will execute the flow and redirect to the dataset module

  import pandas as pd
