It contains the following operations like data connection, dataset and dataflow module etc.

What is data connection?â€‹

Data connection utility help user to create connections for data sources like pull from URL, MSSQL, MySQL, PostgreSQL, SFTP, Snowflake, Amazon s3 and Google bigquery options. You can automate the data connection using scheduler activity. It will pull the data after every minute, hour, day and week based on scheduler selection in specific data connection. In data connection module you can see only raw files.

Data Sourcesâ€‹

what is data source?â€‹

In predictsense data source is medium to upload or fetch data from different sources like URL, MSSQL, MySQL, PostgreSQL, Snowflake, Amazon s3, SFTP, Google Bigquery etc.

Uploadâ€‹

we can upload file using upload option from datasources popup window and fill the required details like file encoding, browse the file from local machine etc.

URLâ€‹

Data can pulled directly from URL with JSON or CSV format. Also, you can schedule the pull activity.

Steps to create a connection using URLâ€‹

  1. Enter connection name
  2. Select resource type, folder name, scheduler
  3. Pass URL and checked the pull data checkbox
  4. click on create button

Databasesâ€‹

Data can be query from existing database and used to make predictions. Also you can schedule the pull from MSSQL, MySQL, PostgreSQL, Snowflake and Google Bigquery etc. To fetch data from databases first need to configure the connections in settings.

Steps to configure the MSSQL, MySQL and PostgreSQL connectionsâ€‹

  1. Goto settings->connections->databases
  2. Enter the connection name
  3. Enter database address and port number
  4. Enter datbase username, password and predictsense password
  5. Click on test connection button to verify the connection status is valid or invalid
  6. click on create button to configure the setting

Steps to configure the snowflake connectionâ€‹

  1. Goto settings->connections->snowflake
  2. Enter the connection name
  3. Enter snowflake account name, username, password
  4. Enter predictsense password
  5. Click on test connection button to verify the connection status is valid or invalid
  6. click on create button to configure the setting

Steps to configure the bigquery connectionâ€‹

  1. Goto settings->connections->bigquery
  2. Enter the connection name
  3. Enter predictsense password
  4. Enter google cloud project id, client email, service account key
  5. Click on test connection button to verify the connection status is valid or invalid
  6. click on create button to configure the setting

Steps to create a connection using databaseâ€‹

  1. Goto data->data connection
  2. Enter connection name
  3. Select connection
  4. Enter databse name and select query
  5. Enter predictsense password
  6. Click on create button

SFTPâ€‹

SFTP stand's for Secure File Transfer Protocol. In predictsense you can pull data from sftp server only xls, xlsx and csv file format. To fetch the data using sftp first need to configure the sftp settings.

Steps to configure the SFTP settingâ€‹

  1. Goto the settings -> Connections->SFTP
  2. Enter the connection name
  3. Enter the address 127.0.0.1
  4. Add port number 22
  5. Add sftp username and password
  6. Enter the predictsense password
  7. click on Test connection button to check connection status is valid or invalid
  8. click on create button

Steps to create a connection using SFTP settingâ€‹

  1. Enter the connection name
  2. select the connection
  3. select folder path
  4. Select the file extension from dropdown
  5. click on create button

Amazon S3â€‹

Amazon S3 provides management features so that you can optimize, organize, and configure access to your data to meet your specific business, organizational, and compliance requirements.

To fetch the data from S3 first need to configure the connection in settings

Steps to configure the amazon s3â€‹

  1. Goto settings->Connections->Amazon S3
  2. Enter the connection name
  3. Enter AWS region, access key id, secret key
  4. Enter predictsense password
  5. Click on test connection button to check the connection status is valid or invalid
  6. Click on create button

Steps to create a connection using s3â€‹

  1. Goto->data->data connection
  2. Enter the connection name
  3. Select connection and bucket
  4. Enter bucket folder path
  5. Select file extension
  6. click on create button

Create folderâ€‹

It is used to organize different data connections and files in multiple folders. You can create it while creating a connection or by clicking the Create Folder button in the data connection module.

Sample datasetsâ€‹

It provides the various datasets based on project type like predictive modeling, clustering and timeseries. User can import it and use it for further processes like eda, training and model evaluation etc.

Steps to import the sample datasetsâ€‹

  1. Goto data->data connection
  2. Click on sample datasets button it will shows the datasets details on card's based on project type
  3. Click on import button
