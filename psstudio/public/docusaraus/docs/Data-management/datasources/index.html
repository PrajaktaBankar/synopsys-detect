A data set is a collection of numbers or values that relate to a particular subject. For example, the test scores of each student in a particular class is a data set. In predictsense data uploaded from different sources and connections are listed over here. All the listed files are structured and ready for further process like eda, model training and model evaluation etc.

It provide the following dataset actions like preveiw, detailed report, merge and delete dataset etc.

A dataset preview shows the first 5 records only from dataset into table view.

Merge data setâ€‹

Merge Dataset feature provide the option to merge two different datasets based on common column in both the datasets.

- Row wise merge
- Column wise merge

Row wise mergeâ€‹

Merging two datasets is the process of bringing two datasets together into one, and aligning the rows from each based on common attributes.

Column wise mergeâ€‹

It is the process of bringing two datasets together into one, and aligning the columns from each based on common attributes. It provides the diffrent operations like inner, outer, left and right join etc.
