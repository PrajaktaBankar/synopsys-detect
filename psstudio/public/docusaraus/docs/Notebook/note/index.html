What is notebook?

In PredictSense notebook allows user to write custom code for analysis or data transformation.

Steps to create notebook​

  1. Goto data connection
  2. Upload dataset
  3. Click on notebook in left side menu bar.
  4. Click on Create or create now. button
  5. Enter the notebook name, select input file and select folder.
  6. Click on Create button. It will create the notebook and list in the table form.

Notebook functionalities​

Notebook is equipped with a python-based utility to interact with the PredictSense system. And on creating a new notebook, some cells are populated with starter code for the user to start interacting with PredictSense.

Reading data​

The user can use the list_uploaded_datasets API to list the available files. The load_pandas_dataframe helps to load the selected file.

Data analysis​

With the help of VisualAnalysis API, the user can also generate a set of interactive data analysis widgets to inspect and analyse the relationships in the data.

Create shareable reports​

save_report API allows user to save plots or dataframes to the reports module. These reports then can be shared with different users.

Share function​

PredictSense’s share_function utility allows user to collaborate with other users via feature sharing functionality. A function can be shared with other users by providing the function and the intended email address.

Load shared function​

A shared feature can be utilized by loading it in memory with get_shared_function API.

Save data​

Once the custom data transformation is done, the user can use this engineered data to use for further processing in PredictSense by calling the “save_data” API. This will register the newly created data to the data pages and hence allow user to perform EDA and training.
