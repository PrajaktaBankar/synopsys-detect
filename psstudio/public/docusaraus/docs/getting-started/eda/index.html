Exploratory Data Analysis (EDA)

Before we move on to the model building, we need to analyze our data and make sure it is clean.

Target Selectionâ€‹

We need to select the data and the dependent (target) variable.

In case of insurance dataset, we select insurance_claim as the target. Once the target is selected, we see a bar chart explaining the distribution of the variable. The insurance_claim is a binary variable, 1 indicating that a claim was made.

Descriptive Statistics & Data Cleaningâ€‹

PredictSense automatically identifies the data type of each variable and computes the descriptive statistics. We can change the data type, imputation strategy and outlier detection & handling strategies as per our need.

For our insurance_claim prediction, we do not seem to have any missing values and the data types also seem accurate. We will go with the default outlier strategies for now.

We can now start building our models!
