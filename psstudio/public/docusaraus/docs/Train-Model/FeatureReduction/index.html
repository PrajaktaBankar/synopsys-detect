Feature reduction

Features reduction are done by selecting best features and applying dimensionality reduction techniques. Feature reduction, also known as dimensionality reduction, is the process of reducing the number of features in a resource heavy computation without losing important information. Reducing the number of features means the number of variables is reduced making the computerâ€™s work easier and faster. Feature reduction can be divided into two processes.

  1. Feature Selection
  2. Feature Extraction

Feature selection methodâ€‹

Feature selection is a technique where we choose those features in our data that contribute most to the target variable.

Types of feature selection methodsâ€‹

  1. SelectKBest
  2. SelectPercentile
  3. RFE

SelectKBestâ€‹

Select features according to the k highest scores.

SelectPercentileâ€‹

Select features according to a percentile of the highest scores.

RFEâ€‹

Recursive feature elimination (RFE) is a feature selection method that fits a model and removes the weakest feature (or features) until the specified number of features is reached.

Scoring Functionâ€‹

Based on the selected method score for each feature is calculated.

Types of Scoring Functionâ€‹

  1. chi2
  2. f_classif
  3. mutual_info_classif
  4. f_regression
  5. mutual_info_regression

chi2â€‹

The chi2(Ï‡2) test is used in statistics to test the independence of two events. More specifically in feature selection we use it to test whether the occurrence of a specific term and the occurrence of a specific class are independent.

f_classifâ€‹

Used only for categorical targets and based on the Analysis of Variance (ANOVA) statistical test.

mutual_info_classifâ€‹

Used only for categorical targets and based on the Analysis of Variance (ANOVA) statistical test.

f_regressionâ€‹

The F value in regression is the result of a test where the null hypothesis is that all of the regression coefficients are equal to zero. Basically, the f-test compares your model with zero predictor variables (the intercept only model), and decides whether your added coefficients improved the model.

mutual_info_regressionâ€‹

Mutual information between two random variables is a non-negative value, which measures the dependency between the variables. It is equal to zero if and only if two random variables are independent, and higher values mean higher dependency.

Calculate feature scoreâ€‹

Calculate the relationship between each input variable and the target variable using statistics and selecting those input variables that have the strongest relationship with the target variable.

What is dimensionality reduction?â€‹

Dimensionality reduction is the process of reducing the number of random variables under consideration, by obtaining a set of principal variables. It can be divided into feature selection and feature extraction.

Dimensionality reduction optionsâ€‹

  1. PCA
  2. Factor Analysis

PCAâ€‹

Principal Component Analysis(PCA) is one of linear dimension reduction. PCA is a projection based method which transforms the data by projecting it onto a set of orthogonal axes.

Factor Analysisâ€‹

Factor analysis is a technique that is used to reduce a large number of variables into fewer numbers of factors.
